# CineList-Movie-WatchList-App
A full-featured movie watchlist app. Search for any film, explore detailed info, write reviews, and curate your personal watchlist — all in one place.

- Movie search — query the full TMDB catalogue with live loading and error states
- Movie detail — poster, rating, genres, overview, and release info on a dedicated page
- Watchlist management — add and remove movies, persisted in Redux state
- Review system — star rating (1–5), written review, and date watched per movie
- Filter and sort — filter your watchlist by genre, sort by rating or date watched
- Form validation — review form validated with Yup and managed with React Hook Form
